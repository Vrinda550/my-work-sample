# 🚀 Dynamic Task Tracker Dashboard

A clean, responsive, and interactive data management dashboard built using modern semantic **HTML5, CSS3, and vanilla JavaScript (ES6)**. This project serves as a professional technical work sample for frontend internship opportunities, demonstrating foundational capabilities in responsive layouts, state management, and DOM manipulation.

## ✨ Key Features

*   **Live Counter Metrics:** Automatically tracks and displays the real-time count of total tasks vs. completed tasks.
*   **Custom Category Tags:** Includes an intuitive dropdown selector to quickly organize and assign tasks under specific categories (*College*, *Personal*, or *Other*).
*   **Interactive DOM Handshakes:** Users can dynamically check off tasks to visually strike them through as completed, or permanently remove them using the delete action.
*   **Keyboard Accessibility:** Supports event listeners that allow submitting new tasks instantly by pressing the "Enter" key.
*   **Modern CSS UI Layout:** Fully styled using dynamic CSS variables, flexible layouts via Flexbox, smooth hover transitions, and custom scrolling behaviors.

---

## 🛠️ Tech Stack Used

*   **HTML5:** Structured semantic markup, dynamic input controls, dropdown selects, and data lists.
*   **CSS3:** Styled with custom root variables, CSS Flexbox adjustments for layout structure, micro-interactions, and visual transitions.
*   **JavaScript (ES6):** Native DOM selection (`getElementById`), object array state management, event listeners, and dynamic UI rendering using array methods (`.push()`, `.splice()`, `.filter()`).

1. Clone or download this repository to your local computer.
